# **Hybrid Athlete Master Blueprint**

This comprehensive training protocol integrates absolute strength progression, high-volume calisthenics, and advanced aerobic engine building to achieve peak performance across multiple modalities concurrently.

## **1\. Performance Objectives**

* **October Peak:** 100kg Bench Press, 120kg Squat, 1:25 Half Marathon.  
* **November Peak:** 125 Push-ups, 35 Pull-ups, 9:30 3k, 55-second 400m.

## **2\. The Optimized AM/PM Split**

| Day | AM Protocol (Morning) | PM Protocol (Evening)   |
| :---- | :---- | :---- |
| **Sunday** | Passive Rest / Sleep in | Mixed Practice (Sprints, VO2 Max, Calisthenics) |
| **Monday** | Easy Run (Leg flush from Sunday) | Lower Body A (Heavy Absolute Strength) |
| **Tuesday** | Medium Long Run \+ 6x20s Strides | Upper Body A (Heavy Push/Pull \+ Dropset) |
| **Wednesday** | Threshold Run (The 1:25 HM Engine) | Lower Body B (Speed/Hypertrophy \+ Core) |
| **Thursday** | Easy Run (Active Recovery) | Upper Body B (Volume/Capacity) |
| **Friday** | Long Run (Endurance Focus) | Mobility, Injury Prevention, & Sauna |
| **Saturday** | Complete Rest / Light Flush Movement | Complete Passive Rest |

## **3\. Strength & Calisthenics Protocol**

### **Lower Body A (Monday) \- Absolute Strength**

* **Pogo Hops:** 3 sets  
* **Squats:** 3 sets x 3-5 reps (Heavy, driving toward 120kg)  
* **RDL:** 3 sets (Heavy)  
* **Adductor Machine:** 2 sets  
* **Standing Calf Raises:** 3 sets  
* **Bent Knee Calf Isometric:** 2 sets  
* **Long Lever Single Leg Glute Bridge Isometric:** 2 sets  
* **Copenhagen Plank:** 2 sets  
* **Core:** Hanging Leg Raises / Toes-to-Bar (3 sets)

### **Upper Body A (Tuesday) \- Heavy Push/Pull & Fatigue Squeeze**

* **Bench Press:** 4 sets x 3-5 reps (Heavy, driving toward 100kg) ➔ *Immediately perform 1 max-effort set of bodyweight push-ups after the final set.*  
* **Weighted Pull-ups:** 3 sets (Heavy) ➔ *Immediately perform 1 max-effort set of bodyweight pull-ups after the final set.*  
* **Incline Smith Machine Press:** 3 sets  
* **Rows:** 2 sets  
* **Lateral Raises:** 3 sets  
* **Rear Delt Flies:** 2 sets  
* **Tricep Extensions & Preacher Curls:** 3 sets each

### **Lower Body B (Wednesday) \- Speed & Power**

* **Depth Jumps:** 3 sets  
* **Squats:** 3 sets x 8-10 reps (Moderate weight, fast bar speed)  
* **Lying Leg Curl:** 3 sets  
* **Adductor & Glute Kickback Machines:** 2 sets each  
* **Bent Knee Calf Isometric:** 3 sets  
* **Core 1:** Ab Wheel Rollouts (3 sets for squat bracing)  
* **Core 2:** Heavy Suitcase Carries (3 sets per side for anti-lateral flexion)

### **Upper Body B (Thursday) \- Absolute Ceiling**

* **Weighted Dips:** 3 sets (Heavy) ➔ *Immediately perform 1 max-effort set of bodyweight push-ups after the final set.*  
* **Weighted Pull-ups:** 3 sets (Heavy) ➔ *Immediately perform 1 max-effort set of bodyweight pull-ups after the final set.*  
* **Overhead Barbell Shoulder Press:** 3 sets  
* **Rows & Lateral Raises:** 2 sets & 3 sets  
* **Shrugs:** 2 sets  
* **Tricep Extensions & Preacher Curls:** 3 sets each

## **5\. Mobility & Recovery Regimen**

* **Nightly Pre-Bed (Parasympathetic):** 5-8 minutes of gentle Couch Stretch, 90/90 Hip Sit, Downward Dog, and Child's Pose to lower core temperature and improve sleep architecture.

## **6\. Fueling Architecture (150g Protein / 350g Carbs)**

1. **Pre-AM Run:** 30-50g simple carbs (banana/electrofuel) \+ Coffee  
2. **Post-AM Run (Reload):** Massive carbohydrate hit (oats/rice/sourdough) \+ 3-4 whole eggs. Take Vitamin D3+K2.& Omega 3  
3. **Mid-Day (Sustained Engine):** Lean beef/steak, complex carbs, and leafy greens.  
4. **PM Pre-Lift:** 30-60 minutes prior, consume electrofuel or fruit. Take Beta-Alanine.  
5. **PM Post-Lift (Rebuild):** High protein (beef/poultry) and easily digestible carbs. Add Creatine to the shake/water.  
6. **Pre-Bed:** Magnesium Glycinate, Zinc and Copper 30-60 minutes before sleep.
