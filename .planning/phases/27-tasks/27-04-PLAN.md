parseTaskInput contract (RESEARCH Unknown 1):
```
interface ParsedTask { title: string; due_date: string|null; list_name: string|null; priority: 'none'|'low'|'medium'|'high'|null }
parseTaskInput(raw: string, refDate?: Date): ParsedTask
PRIORITY_MAP: !high/!1→high, !medium/!2→medium, !low/!3→low
```
Strip #list and !priority tokens BEFORE chrono.parseDate; resolve date with {instant: refDate, timezone:'Asia/Jerusalem'}; format YYYY-MM-DD via toLocaleDateString('en-CA',{timeZone:'Asia/Jerusalem'}); unparsed → due_date null, list_name null.
