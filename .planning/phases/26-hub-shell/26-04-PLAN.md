## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → `GET /api/today` | Authenticated read; gated by `require_hub_session` (HUB-01) — no unauthenticated data exposure |
| `/api/today` → external tools (Routes/Garmin/weather) | Server-side egress; results cached in-process; never exposes raw upstream errors to the client |
| Firestore docs → JSON response | `DatetimeWithNanoseconds` must be ISO-converted or the response 500s |

| T-26-04-02 | Denial of Service | Sync tool calls blocking the single-worker event loop, stalling all hub requests | mitigate | All sources via `run_in_executor` + `asyncio.gather` (Pitfall 2); per-helper try/except so one slow/failed source can't hang the whole route |
